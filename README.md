# react-video-app

A simple mobile video app for ts+react+antD+sass built through vite

# run ths project

```
pnpm run dev
```

# 项目介绍

## 项目根目录

```
├─ src
├─api // 请求相关
├─assets // 图片以及 svg 相关
├─style // 公共样式
├─components // 抽象出可复用的 UI 组件
├─utils // 工具类函数和相关配置
├─types // ts 相关类型约束文件
├─views // 页面
├─router // 路由配置文件
└─store // redux 相关文件
App.tsx // 根组件
main.tsx // 入口文件

```
